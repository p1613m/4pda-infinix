Для предпросмотра нужно открыть файл index.html в браузере.

HTML-код для вставки в разлел находится в файле html.txt (тот код, который как раз вставляется в блок .rnd-replace-max-height-350)
В HTML-коде присутствуют комментарии, которые помогут разместить контент

CSS-код для вставки в раздел находится в файле css.txt, если нужен такой же но минифицированный код, то он в файле css.min.txt

CSS-код для блоков в сайтбар находится в файле css-sidebar.txt, если нужен такой же но минифицированный код, то он в файле css-sidebar.min.txt

Суффикс для CSS-класса сайтбаров: infinix-sidebar
Код для каждого сайтбара находится в папке sidebar

Данные файлы и папки находятся в папке for-site

Подготовленные оптимизированные изображения находятся в папке images
